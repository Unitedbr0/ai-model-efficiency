# AI Model Efficiency Analysis 📊

This project analyzes the performance of different AI models using key metrics like **Accuracy**, **Latency**, and **Efficiency**.

---

## 🚀 Project Overview

In real-world AI systems, choosing the best model is not just about accuracy — **speed (latency)** also matters.

This project helps compare models using:

* Accuracy 📈
* Latency ⏱️
* Efficiency ⚡ (Accuracy / Latency)

---

## 📈 Output

### 1. Model Accuracy Graph

### 2. Model Efficiency Graph

Efficiency is calculated as:

```

```

This helps identify the best balance between performance and speed.
